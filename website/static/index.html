Installation

Download with cURL, grant permissions, and move to your PATH.

# Download the latest binary (replace with actual release URL)
curl -L -o zaptun https://github.com/harsh082ip/ZapTun/releases/latest/download/zaptun-linux-amd64

# Make it executable
chmod +x zaptun

# Move to a directory in your PATH for global access
sudo mv zaptun /usr/local/bin/zaptun

Use PowerShell to download the executable.

# Download using PowerShell (replace with actual release URL)
Invoke-WebRequest -Uri "https://github.com/harsh082ip/ZapTun/releases/latest/download/zaptun-windows-amd64.exe" -OutFile "zaptun.exe"

# We recommend moving zaptun.exe to a folder and adding it to your System PATH.

Usage

Authenticate

First, obtain an authentication token from our auth server. You can get one by visiting this link. Once you have the token, authenticate the CLI.

# Replace  with the token you received
zaptun auth <your-auth-token>

Start a Tunnel

Expose a local web server (e.g., running on port 3000) to the internet.

# Expose local port 3000 via HTTP
zaptun http 3000

You can also expose any TCP service, such as SSH.

# Expose local port 22 via TCP for SSH
zaptun tcp 22
